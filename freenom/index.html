Freenom

→ freenom.com öffnen

→ Wunschdomain mit .tk-Endung eingeben

→ Checkout (oben rechts) wählen

→ «Forward this Domain» (beliebige URL eingeben)

→ 12 Monate einstellen → Verify via Mail

→ Link in Mail klicken (kann einige Minuten dauern) → Anmeldung ausfüllen & Login

→ Unter «My Domains» → «Manage Domain»

→ Management Tools
→ Use default nameservers (Freenom Nameservers) → Manage Freenom DNS
→ IP in DNS eintragen
